# Publicar nova base da Wiki

Sempre que voce atualizar o `data/WikiItems.json`, rode:

`scripts\publish-wiki-data.bat`

O script vai:
- adicionar os arquivos principais
- pedir uma mensagem de commit
- fazer o `push` para o GitHub

Depois disso, quem usa a extensao so precisa clicar em:

`Atualizar base`
